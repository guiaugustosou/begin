Elementos, Formatação e Estrutura HTML

Além da estruturação já conhecida do HTML, a iniciar com a declaração DOCTYPE, seguida de html, head, body, há inúmeros recursos de formatação e estilização que serão abordados a seguir.

1. "background"

Diversos aspectos do fundo da página podem ser alterados pela propriedade CSS background. Vejamos as funções.

1.1. background-color

Define a cor de fundo de um elemento.

Sintaxe:

background-color:valor

seletor: { background-color:#CCC; }

Aplicação no CSS interno:

< ! DOCTYPE html >

< html lang= "pt-br" >

< head >

< title > Título da Página < /title>

< s tyle >

body { background-color: #CCC; }

< / style >

< / head >

1.2. background-image

Define a imagem de fundo de um elemento. Por padrão, sempre que utilizamos a propriedade background-image, o navegador repetirá a imagem quantas vezes for necessário até preencher todo o fundo do elemento.

Sintaxe:

background-image:url("nome ou endereço da imagem")

seletor: { background-image:url("imagem.gif"); }

No método CSS interno, fica da seguinte forma:

< ! DOCTYPE html >

< html lang= "pt-br" >

< head >

< title > Título da Página < /title>

< s tyle >

body { background-color: url("image.gif"); }

< / style >

< / head >

1.3. background-repeat

Define como a imagem se repetirá no fundo de um elemento.

Sintaxe:

seletor {background-image:url("nome_da_imagem.jpg");}

body {background-image:url("nome_da_imagem.jpg");}

1.3.1. background-repeat:repeat

A imagem é repetida até cobrir todo o fundo. A última imagem será cortada se não couber.

body {background-image:url("nome_da_imagem.jpg"); background-repeat:repeat;}

1.3.2. background-repeat:no-repeat

A imagem não se repete.

body {background-image:url("nome_da_imagem.jpg"); background-repeat:no-repeat;}

1.3.3. background-repeat:repeat-x

A imagem se repete na horizontal (eixo x)

body {background-image:url("nome_da_imagem.jpg"); background-repeat:repeat-x;}

1.3.4. background-repeat:repeat-y

A imagem se repete na vertical (eixo y)

body {background-image:url("nome_da_imagem.jpg"); background-repeat:repeat-y;}

1.3.5. background-repeat:space

A imagem se repete sem recortes e com distribuição de espaçoes em branco entre elas para que não haja recortes da imagem

body {background-image:url("nome_da_imagem.jpg"); background-repeat:space;}

1.3.6. background-repeat:round

A imagem se esticará ou comprimirá para se adaptar a tela.

body {background-image:url("nome_da_imagem.jpg"); background-repeat:round;}

1.4. background-attachment

Define se uma imagem de fundo é fixa ou rola com o resto da página. Esta propriedade aceita como valores: a) fixed, que deixa o fundo fixo e não rola quando o usuário mexe no scroll da página; b) scroll, que também rola o fundo ao utilizar o scroll da página.

Sintaxe:

body {background-image:url("nome_da_imagem.jpg"); background-attachment:fixed;}

body {background-image:url("nome_da_imagem.jpg"); background-attachment:scroll;}

1.5. background-position

Define a posição inicial de uma imagem de fundo. Os valores admitidos são os seguintes:

a) left top, left center, left bottom, right top, right center, right bottom, center top, center center, center bottom - para posicionamentos fixos;

b) x% y% - para posicionamento no eixo x (horizontal) e eixo y (vertical), conforme a porcentagem/=;

c) xpos ypos - para posicionamento no eixo x (horizontal) e eixo y (vertical, com medida em pixels;

d) initial - é valor padrão (esquerda superior)

e)inherit - toma a propriedade do elemento pai

Sintaxe:

body {background-image:url("nome_da_imagem.jpg"); background-repeat:no-repeat; background-attachment:fixed; background-position: center;}

1.6. background-size

Define o tamanho da imagem de fundo. Admitem-se os seguintes valores:

a) auto - valor padrão (mesma largura e altura da imagem);

b) lenght:width height - define um valor em pixel para a largura e altura. Se apenas um é definido, o segundo fica como auto;

c) % - define um valor em porcentagem para a largura e altura. Se apenas um é definido, o segundo fica como auto;

d) cover - encaixa a imagem de fundo, preenchendo todo o espaço, sem repetição (pode ser que a imagem não apareça toda)

e) contain - encaixa a imagem perfeitamente no fundo da página;

f) initial: valor padrão (mesma largura e altura da imagem)

g) inherit - herda a propriedade do elemento pai

Sintaxe:

body {background-image:url("nome_da_imagem.jpg"); background-repeat:no-repeat; background-size: 300px 500px;}

1.7. background-origin

Define como a imagem de fundo está posicionada. Os valores aplicáveis são os seguintes:

a) border-box - a imagem inicia no canto superior esquerdo e ocupa a área da borda (se houver);

b) padding-box - imagem inicia no canto superior esquerdo e desconsidera a área da borda (se houver)

c) content-box: imagem inicia no canto superior esquerdo e desconsidera a área da borda e do padding (espaçamento interno, se houver).

Sintaxe:

body {background-image:url("nome_da_imagem.jpg"); background-repeat:no-repeat; padding:50px; background-origin:padding-box;}

Obs: a propriedade padding:valor formata espaçamento interno.

1.8. background-clip

Define a área de pintura do plano de fundo. As propriedades aplicáveis são:

a) border-box: começa do canto superior esquerdo e preenche todo o fundo incluído a borda (padrão);

b) padding-box: começa do canto superior esquerdo e não preenche a borda (padrão);

c) content-box - preenche apenas a área com conteúdo;

d) initial: define a propriedade como padrão (border-box).

Sintaxe:

body {border:20px black; padding:55px; background:yellow; background-clip:content-box;}

1.9. Declaração resumida do background

É possível sintetizar (reduzir) o tamanho do comando background e suas propriedades, conforme a seguinte ordem:

seletor {background: background-color background image background-repeat background attachment background position;}

Obs: se algum valor for omitido, toma-se o padrão.

Sintaxe:

body {background: #f00 url("fundo.jpg") no-repeat fixed 200px 150px;}

1.9. Múltiplos Fundos

O CSS3 permite aplicar mais de uma imagem de fundo a um elemento. Ocorre uma sobreposição de fundos, conforme a ordem dos comandos na declaração. As propriedades de todas essas imagens é separada por vírgula. Vejamos a sintaxe:

body {background-image: url(img1.gif), url(img2.gif), url(img3.jpg); background-position: right bottom, left top, center top; background-repeat: no-repeat, repeat, no-repeat;}

2. "border"

A borda de um elemento HTML é trabalhada pela propriedade border. Ela admite os seguintes valores:

a) border-width - define a largura da borda. O valor padrão é "medium" (médio)

b) border-style - define o estilo da borda. O valor padrão é "none" (nenhum).

c) border-color - define a cor da borda. O valor padrão é a cor do elemento definido.

d) inicial - define esta propriedade como seu valor padrão;

e) inherit - herda essa propriedade de seu elemento pai

Sintaxe resumida:

seletor {border-width border-style border-color;}

h1 {border: 3px solid #F00;}

< head >

< title > Page Title < / title >

< s tyle >

body { border : 10 px solid #FF0 ; }

h1 { border : 3 px solid #F00 ; }

p { border : 3 px solid #0F0 ; }

< / style >

< / head >

3. "margin" e "padding"

A margem e o espaçamento interno poderão ser modificados pelas propriedades "margin" (fomata o espaço da borda para fora) e "padding" (formata o espaço da borda para dentro). O atributo border fica entre a margin e o padding.

Margin aceita os seguintes valores:

a) margin-top - margem superior;

b) margin-right - margem da direita

c) margin-left - margem da esquerda

Padding aceita os seguintes valores:

a) padding-top - espaçamento superior;

b) padding-right - espaçamento da direita;

c) padding-bottom - espaçamento inferior;

d) padding-left - espaçamento da esquerda

3.1. Forma abreviada

Conforme o sentido horário, podem ser inseridos até quatro valores na fórmula abreviada de margin, sendo o primeiro valor referente a superior, em seguida a da direita, inferior e esquerda:

margin: 15px 10px 20px 20px;

Se forem apenas dois valores, eles referir-se-ão ao mesmo eixo, isto é, às margens superior e inferior; e direita e esquerda:

margin: 15px 20px

Por fim, se houver apenas um valor, ele será atribuído às quatro margens:

margin: 12px

Obs: valores negativos são permitidos para margin, mas não para padding.

4. "font", "text" e "line"

Os elementos textuais são modificados pelas tags font, text e line.

4.1. Font

4.1.1. Font-family

Formata a familia da fonte aplicada a um elemento. Você pode declarar até três fontes separadas por vírgulas, sendo que fontes que possuem espaços em seu nome devem vir dentro de aspas.

Sintaxe:

p { font-family: "Times New Roman", Georgia, Serif; }

4.1.2. Font-size

Especifica o tamanho de uma fonte aplicada a um elemento

Sintaxe:

h1 { font-size: 120%; }

h2 { font-size: 30px; }

p { font-size: 2em; }

4.1.3. Font-style

Especifica os estilos italic, oblique ou normal para o elemento

Sintaxe:

p { font-style: normal; }

h1 { font-style: italic; }

h2 { font-style: oblique; }

4.1.4. Font-variant

Define a fonte como minúsculas ou Versalete (maiúsculas).

Sintaxe:

p { font-variant: small-caps; }

4.1.5. Font-weight

Define o peso da fonte, tais como bold (negrito).

Sintaxe:

p { font-weight: normal; }

.teste { font-weight: bold; }

4.2. Line

4.2.1. Line-height

Define a altura de uma linha.

Sintaxe:

p { line-height:20px; }

4.3. Text

4.3.1. Text-align

Alinha os textos na horizontal no centro (center), esquerda (left) e direita (right)

Sintaxe:

h1 { text-align: left; }

p { text-align: center; }

h2 { text-align: right; }

4.3.2. Text-decoration

Define a decoração do texto para os elementos. Podem ser none (nenhum), underline (linha de baixo), overline (linha de cima) ou line-through (linha através).

Sintaxe:

h1 { text-decoration: overline; }

4.3.3. Text-ident

Recua a primeira linha, criando o recuo de um parágrafo

Sintaxe:

p { text-indent: 30px; }

4.3.4. Text-transform

Transforma o texto em uppercase (maiúscula), lowercase (minúscula) e capitalize (primeira letra de cada palavra em maiúscula). O valor none também é aplicável.

Sintaxe:

.minuscula { text-transform: uppercase; }

.maiuscula { text-transform: lowercase; }

.primeira_letra { text-transform: capitalize; }

4.4. "@font-face"

Permite apontar uma fonte fora do banco de fontes Websafe da máquina. Neste caso primeiro se indica qual o nome da fonte com font-family e logo em seguida qual sua localização com src:url(localização). Lembrando: src é abreviação de source ou origem em português.

Sintaxe:

@font-face { font-family: minhafonte; src:url(teste_fonte.woff); }

Obs: o nome da fonte deve sempre ser minúsculo.
